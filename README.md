What is it?
===========

A Puppet module that ensures EC2 ephemeral disks are mounted.

Released under the Apache 2.0 licence

Usage:
------

To mount the disks:
<pre>
  include aws_mount
</pre>


Known Issues:
-------------
* 4 and 24 disks not supported at this time
